# FlowDesk – AI Productivity Assistant

FlowDesk is an AI-powered workplace assistant designed to help users save time and improve productivity by automating everyday business tasks. It combines multiple smart tools into one simple and easy-to-use platform.

--------------------------------------------------

FEATURES

1. Email Generator
- Creates professional emails instantly
- Allows tone customization (formal, friendly, persuasive)
- Ready-to-copy output

2. Meeting Notes Summarizer
- Turns long notes into short summaries
- Highlights key points, decisions, and action items

3. Task Planner
- Helps plan daily activities
- Organizes tasks by priority
- Improves time management

4. Research Assistant
- Quickly finds and summarizes information
- Simplifies complex topics
- Saves time on research

--------------------------------------------------

PURPOSE

FlowDesk is built to:
- Reduce repetitive work
- Increase efficiency
- Provide quick AI assistance
- Help users stay organized

--------------------------------------------------

HOW IT WORKS

1. Select a tool (Email, Notes, Tasks, or Research)
2. Enter your input or prompt
3. Receive instant AI-generated results
4. Copy or edit the output as needed

--------------------------------------------------

EXAMPLE USE CASES

- Writing emails quickly
- Summarizing meetings
- Planning your day
- Doing research for assignments or work

--------------------------------------------------
